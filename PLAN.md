# Mate

> Your personal library of things worth your time.

## What is Mate?
Mate is a smart bookmarking tool that lets you save anything from the web,
automatically summarised and categorised by AI. Built for people who consume
a lot of content and want to actually remember it.

## Stack
- **Frontend** Next.js + Tailwind CSS
- **Database + Auth** Supabase
- **AI** Claude API
- **Deployment** Vercel

## V1 Scope
The goal is simple: prove the core loop works.

- [ ] Project setup (Next.js + Supabase + Vercel pipeline)
- [ ] Auth (email/password via Supabase)
- [ ] Save a URL manually (paste + submit)
- [ ] AI summary + category generated on save
- [ ] Basic UI to view saved items
- [ ] Folders to organise items (Arc-like)

## Out of scope for V1
- Browser extension
- iOS shortcut
- YouTube transcription
- Sharing / public profiles

## V2 Ideas
- Browser extension (one-click save)
- iOS shortcut
- YouTube + podcast transcription
- Public profiles and sharing
- Import from Raindrop, Pocket
